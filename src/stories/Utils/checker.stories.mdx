### clearClassName(value?: string)

- className에 빈 값이 있으면 undefined가 아닌 empty string으로 반환해주는 function
- @param {string} value
- @return {string}

### 예제코드

```
  // className is ''
  <div className={clearClassName('')} />
  <div className={clearClassName(undefinded)} />

  // className is 'test'
  <div className={clearClassName('test')} />

```

### isIE()

- 브라우저가 IE 인지 체크하기 위한 function
- @return {boolean}

### 예제코드

```
  <div style={{ fontSize: isIE() ? '14px' : '16px' }} />
```
